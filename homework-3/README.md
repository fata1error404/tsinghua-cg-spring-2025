# Star Rotation

1. Multiple stars rotate around a center.
2. The colors of stars are different.
3. Stars rotate and spread out in spiral path.
4. Choose a spiral type in Archimedes spiral, Fermat spiral and logarithmic spiral. (See https://en.wikipedia.org/wiki/Archimedean_spiral, https://en.wikipedia.org/wiki/Fermat%27s_spiral, https://en.wikipedia.org/wiki/Logarithmic_spiral )

# Display of 3D model

1. Load a 3D mesh model and display it on the screen.
2. 4 modes are supported: `wireframe` mode, `vertex` mode, `face` mode, `face and edge` mode.  Mode is switched by keyboard (ZXCV). Under the face mode, the color of each face is different.
3. Rotate and translate the model by keyboard.
4. Change color of wireframes under wireframe mode by keyboard.
